# 📌 Atividade Individual – Backend

Este projeto tem como objetivo identificar problemas em uma base de código fornecida e implementar melhorias seguindo requisitos específicos.

---

## 🎯 Objetivo

A atividade consiste em analisar a estrutura do código e realizar correções e implementações relacionadas a:

* Tratamento de erros
* Relacionamentos entre entidades
* Autenticação com JWT
* Correção de erros de compilação

---

## ✅ Requisitos da Atividade

### 1. Tratamento de Erros

* Adicionar blocos `try-catch`
* Utilizar corretamente os **status codes HTTP**

---

### 2. Relacionamentos entre Entidades

Identificar e descrever os relacionamentos existentes:

* Um para um (1:1)
* Um para muitos (1:N)
* Muitos para muitos (N:N)

> Apenas identificar e citar entre quais entidades ocorrem.

---

### 3. Autenticação com JWT

* Implementar autenticação utilizando **JSON Web Token**
* Garantir que existam rotas protegidas
* Demonstrar uma requisição que exige token

---

### 4. Correção de Erros de Compilação

* Identificar os erros presentes no código
* Corrigir e garantir que a aplicação funcione corretamente

---

## 🎥 Entrega

* A atividade deve ser feita **em dupla**
* Gravar um vídeo de até **2 minutos**
* No vídeo, deve-se demonstrar:

  * As implementações realizadas
  * O funcionamento do sistema

---

## 📤 Submissão

* Publicar o vídeo no **YouTube**

  * Pode ser como *"não listado"*
* Enviar o link para avaliação
* Informar claramente o **nome dos integrantes**

---

## ⚠️ Observações Importantes

* Trabalhos sem identificação dos integrantes serão considerados **plágio**
* Caso seja identificado:

  * Uso de código gerado por IA
  * Cópia entre colegas
    ➡️ A atividade poderá ser **zerada**
